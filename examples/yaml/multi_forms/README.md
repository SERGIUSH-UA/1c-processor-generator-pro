# Приклад: Обробка з кількома формами

Цей приклад демонструє використання кількох форм в одній обробці.

## Структура

- **Форма** (головна, за замовчуванням) - основна робоча форма
- **Настройки** - форма налаштувань

## Особливості

### Окремі handlers_dir для кожної форми

Кожна форма має власну директорію з BSL handlers:
- `handlers_main/` - handlers для головної форми
- `handlers_settings/` - handlers для форми налаштувань

### Відкриття інших форм обробки

Головна форма містить кнопку "Настройки" (F9), яка відкриває форму налаштувань:

```bsl
ФормаНастроек = ПолучитьФорму("ВнешняяОбработка.МультиФормы.Форма.Настройки");
ФормаНастроек.Открыть();
```

### Форма за замовчуванням

У головної форми встановлено `default: true`, тому вона відкривається при запуску обробки.

## Генерація

```bash
python -m 1c_processor_generator yaml --config examples/yaml/multi_forms/config.yaml
# Результат буде в ./tmp/МультиФормы/
```

## Результат

Буде згенеровано:
- `МультиФормы/МультиФормы.xml` - метадані обробки
- `МультиФормы/МультиФормы/Forms/Форма.xml` - метадані головної форми
- `МультиФормы/МультиФормы/Forms/Форма/Ext/Form/Form.xml` - структура головної форми
- `МультиФормы/МультиФормы/Forms/Форма/Ext/Form/Module.bsl` - модуль головної форми
- `МультиФормы/МультиФормы/Forms/Настройки.xml` - метадані форми налаштувань
- `МультиФормы/МультиФормы/Forms/Настройки/Ext/Form/Form.xml` - структура форми налаштувань
- `МультиФормы/МультиФормы/Forms/Настройки/Ext/Form/Module.bsl` - модуль форми налаштувань
